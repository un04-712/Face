# Face AI

这是一个基于 PyQt5 的人脸识别项目，包含人脸信息录入、摄像头采集、图片识别、数据库管理等功能。

## 主要功能

- 人脸信息录入
- 人脸识别
- 数据库管理
- 摄像头拍照与图片上传

## 运行说明

1. 安装依赖
2. 运行 `main.py`
3. 按界面按钮进行人脸录入与识别

## 项目结构

- `main.py`：程序入口
- `my_ui/face_ui.py`：界面文件
- `my_facenet/`：人脸特征与模型相关代码
- `myYolo/`：YOLO 相关训练与推理代码
- `utils/`：通用工具
- `db/`：数据库相关代码
